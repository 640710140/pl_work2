OOP Concepts and Code Examples

1. Code Examples

Java Code:

class Animal {
    Animal() {
        System.out.println("create animal");
    }

    void sound() {
        System.out.println("animal makes a sound");
    }

    protected void finalize() {
        System.out.println("delete animal");
    }
}

class Dog extends Animal {
    Dog() {
        System.out.println("dog");
    }

    @Override
    void sound() {
        System.out.println("dog barks");
    }

    protected void finalize() {
        System.out.println("delete dog");
    }
}

class Cat extends Animal {
    Cat() {
        System.out.println("cat");
    }

    @Override
    void sound() {
        System.out.println("cat meows");
    }

    protected void finalize() {
        System.out.println("delete cat");
    }
}

public class Main {
    public static void main(String[] args) {
        Animal a = new Animal();
        Dog d = new Dog();
        Cat c = new Cat();

        a.sound();
        d.sound();
        c.sound();
    }
}

Java Output:

create animal
dog
cat
animal makes a sound
dog barks
cat meows

C++ Code:

#include <iostream>
using namespace std;

class Animal {
public:
    Animal() {
        cout << "create animal" << endl;
    }

    virtual void sound() {
        cout << "animal makes a sound" << endl;
    }

    virtual ~Animal() {
        cout << "delete animal" << endl;
    }
};

class Dog : public Animal {
public:
    Dog() {
        cout << "dog" << endl;
    }

    void sound() override {
        cout << "dog barks" << endl;
    }

    ~Dog() {
        cout << "delete dog" << endl;
    }
};

class Cat : public Animal {
public:
    Cat() {
        cout << "cat" << endl;
    }

    void sound() override {
        cout << "cat meows" << endl;
    }

    ~Cat() {
        cout << "delete cat" << endl;
    }
};

int main() {
    Animal* a = new Animal();
    Animal* d = new Dog();
    Animal* c = new Cat();

    a->sound();
    d->sound();
    c->sound();

    delete a;
    delete d;
    delete c;

    return 0;
}

C++ Output:

create animal
dog
cat
animal makes a sound
dog barks
cat meows
delete animal
delete dog
delete cat

2. OOP Concepts

Class

ในแต่ละโปรแกรม คลาส Animal, Dog, และ Cat ถูกสร้างขึ้นเพื่อแสดงความเป็นกลุ่มของวัตถุ (objects) ที่มีคุณสมบัติและพฤติกรรม เช่น method sound().

Object, Instance

Object (instance) ของคลาสจะถูกสร้างเมื่อเราเรียก constructor เช่น a = Animal(), d = Dog(), c = Cat(). แต่ละ object จะมีพฤติกรรมและคุณสมบัติที่เฉพาะเจาะจงตาม class ของมัน.

Subclass, Derived Class

คลาส Dog และ Cat เป็น subclass ของ Animal, พวกมันสืบทอดคุณสมบัติและพฤติกรรมจาก Animal แต่สามารถ override เมธอดของตัวเองได้.

Message

การเรียก method เช่น a.sound() คือการส่งข้อความไปยัง object a ให้ทำงานในลักษณะตามที่กำหนดไว้ใน method sound().

Inheritance

Dog และ Cat สืบทอดคุณสมบัติและพฤติกรรมจาก Animal และสามารถเพิ่มเติมหรือปรับปรุงพฤติกรรมใหม่ได้.

Polymorphism

Method sound() ถูก override ในคลาสลูก (Dog, Cat) แสดงถึง polymorphism ทำให้สามารถเรียก method เดียวกันแต่ได้ผลลัพธ์ที่ต่างกันในแต่ละ subclass.

3. Abstract Class

Abstract Class คือคลาสที่ไม่สามารถสร้าง object ได้โดยตรง และมักใช้เป็น blueprint สำหรับ subclass เพื่อให้มีการสืบทอดและ override method ที่จำเป็น.

ในตัวอย่างนี้ ไม่มี abstract class แต่ถ้าเราต้องการให้ Animal เป็น abstract class เราจะทำให้ method sound() เป็น abstract method ที่ต้องถูก override ใน subclass.

Python Example:

from abc import ABC, abstractmethod

class Animal(ABC):
    @abstractmethod
    def sound(self):
        pass

class Dog(Animal):
    def sound(self):
        print("dog barks")

class Cat(Animal):
    def sound(self):
        print("cat meows")
